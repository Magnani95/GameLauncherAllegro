Progetto-Allegro

Programma per la gestione di più giochi da tavolo.
Il programma utilizza la libreria Allegro5; vedere tale documentazione sul sito.
Per aggiungere un gioco o sottomodulo bisogna:
-aggiungere il prototipo della funzione principale del modulo a giochi.h
-aggiungere il puntatore e il nome alla funzione inizializzazione_giochi in inizializzazione.cc.
-aggiungere agli header di ogni file strutt_dati.h poichè contiene i dati da propagare a tutto il programma.
-se il file usa componenti grafiche di allegro, includere anche allegro_header.h
