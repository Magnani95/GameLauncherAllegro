int inserisci ( st_carta *  source,
st_carta *&  destinazione 
)

Inserisce una doppia lista chiusa in coda a un'altra.

I parametri passati si intendono come puntatori alla testa della lista; al suo precedente sarà attaccata la lista source.
Nel caso la source sia una carta singola (ovvero senza linker), viene creata una lista con un elemento. Nel caso la destinazione sia vuota, viene cambiato il puntatore e punterà alla source.
La funzone non effettua nessun controllo di correttezza sulle regole del gioco.

Restituisce
0 in caso di esito positivo
-99 in caso di source passata vuota.
